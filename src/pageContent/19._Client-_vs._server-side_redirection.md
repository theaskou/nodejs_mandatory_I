# Client-side redirection vs. server-side redirection.

Serverside rendering:

- SEO optimering - crawlers kan ikke eksekvere js, så de kan ikke læse siden hvis den ikke er renderet i forvejen. Der er serverside rendering en fordel.
- Sparer ressourcer, fordi filerne ikke skal læses flere gange?
- Mindsker duplikation
- Løser alle CORS problemer


## Learning goals:

* Can recount the number of ways and approaches to redirection.

* Understands the difference between client-side redirection and server-side.
