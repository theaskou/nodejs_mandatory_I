# Welcome to my Node.js reference work.

This site is a reference work of everything I've learned in my **Fullstack with Node.js** class. The class is taught at [KEA - Copenhagen School of Design and Technology](https://kea.dk/en/), by Anders Latif.

For reference, this is the [semester repository](https://github.com/anderslatif/Kea_DAT_Node.js_2025_Spring). 

This site is build on the principles of file-based routing, which automatically maps files to routes, reducing the need for manual route configuration. You can [view the repository here](https://github.com/KeaAndl/mandatory-i-theaskou).

I hope you will find my notes useful.
